    class Entity

    Базовый класс для всех объектов. От него наследуются все остальные объекты: помещения, устройства, виджеты и т.д.

    image_name[RW]

    Имя изображения. Вызывается каждый раз при обновлении страницы для каждого объекта (как правило раз в пять секунд). Файл с этим именем должен присутствовать в папке app/assets/image/entities По умолчанию берёт первый файл из этой папки в следующем порядке:

    ИмяОбъекта.Значение
ИмяОбъекта
ИмяКласса.Значение
ИмяКласса
ИмяКлассаПредка.Значение
ИмяКлассаПредка
... и т.д. для каждого класса-предка

    Перегрузив этот метод, можно задать индивидуальное изображение для каждого объекта.

    Пример¶ ↑

    def image_name
  "custom_image_#{ value.round(0) }"
end

    at_schedule(options = nil, &block) click to toggle source

    задаёт обработчик, вызываемый по расписанию

    Параметры¶ ↑

    • every - интервал. например 1.hour, 30.minutes, 10.seconds

    • at : String - время в формате 'HH:MM'. Можно также указывать массив, например ['10:00', '13:10']

    Пример¶ ↑

    at_schedule(every: 1.day, at: ['10:00', '13:00']) do
  light1.on! delay: 5.minutes
end

def last_indication(value = nil)
  query = indications.limit(1).order('created_at DESC')
  query = query.where(value: value) if value
  query.first
end
